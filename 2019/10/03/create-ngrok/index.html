前言

有個有趣的需求，需要與其他RD整合 web 的部分

但因為自己在本機 localhost 測試方便，沒辦法轉出去給其他人連進來

朋友推薦使用 ngrok，非常簡單，功能強大~

以 Windows 當範例

  • 官方寫得很詳細官方教學

  1. 首先去官網註冊帳號，註冊完成後登入，會在左邊的列表看到 Auth，點進去會看到右邊有自己的 Authtoken，以及下面的指令複製起來

  2. 再來抓執行檔 連結

  3. 解壓縮後，開啟 powershell

  4. 貼上剛剛複製的 authtoken 的指令 ./ngrok authtoken xxxxxxx

  5. 輸入指令，./ngrok http 80，就能開啟來囉！

是不是超級簡單!!!

還可以設定 subdomain 的指令
