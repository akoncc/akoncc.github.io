文章目錄

kitematic 實在超好用阿~
可是，這次踩了一個雷…

就是官方的 docker destop 附帶轉到官方下載 kitematic 的版本是有問題的 (傻眼)

會出現以下的錯誤: context canceled

解決方法

  1. 將原本安裝的 kitematic 移除
  2. 到官方的 github 去下載正確的版本
  3. 解壓縮後，將資料夾丟到 docker 提示的路徑
  4. 完成！開始享受吧

找到這篇解答，非常感謝

https://www.jianshu.com/p/fd651ba3ec6e
